Ця програма зчитує початкове слово ("supermarket") із текстового поля, після чого розрізає його на окремі літери та склади за допомогою команди Substring. Далі вона склеює ці шматочки між собою так, щоб утворилися нові дійсні слова (наприклад: super, market, mark, park, smart), і виводить готовий список на екран. По суті, це генератор, який автоматично створює нові слова з одного великого за принципом аплікації: вирізав потрібні частини та з'єднав їх разом.
